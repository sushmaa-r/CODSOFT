# 🤖 CODSOFT Machine Learning Internship Projects

This repository contains a collection of machine learning projects developed as part of my internship and learning journey, focusing on solving real-world problems using data-driven approaches.

---

## 🚀 Projects Included

### 📩 Spam SMS Detection

* Classifies SMS messages as Spam or Not Spam
* Uses NLP techniques (TF-IDF) and Logistic Regression
* Achieved ~96% accuracy

---

### 💳 Credit Card Fraud Detection

* Detects fraudulent transactions using imbalanced data
* Compared Logistic Regression, Random Forest, Decision Tree
* Selected Logistic Regression based on high recall

---

### 📉 Customer Churn Prediction

* Predicts customer churn using customer data
* Compared Logistic Regression, Random Forest, Gradient Boosting
* Selected Gradient Boosting as final model (~86.7% accuracy)

---

## 🔁 Workflow

Each project follows a structured machine learning pipeline:

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Train-Test Split
5. Model Training
6. Model Evaluation (Accuracy, Precision, Recall, Confusion Matrix)
7. Model Comparison & Selection
8. Model Saving using Pickle
